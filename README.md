# 数据可视化作业展示

这是一个基于Web的数据可视化项目，展示了10种不同类型的图表，每个图表都具备完整的交互功能。

## 项目特点

- 🎯 **10种图表类型**：包含柱形图、条形图、箱线图、折线图、直方图、饼图、气泡图、堆积面积图、误差棒图、雷达图
- 💡 **完全交互**：每个图表都可以实时编辑数据、标签和坐标轴
- 🎨 **美观界面**：现代化的UI设计，响应式布局
- 🔄 **实时更新**：修改数据后立即更新图表显示
- 📱 **移动友好**：适配各种屏幕尺寸

## 图表功能

### 1. 单组柱形图
- 编辑图表标题、Y轴标题
- 修改5个标签名称和数据值

### 2. 多组条形图  
- 编辑X轴标题、数据集标题
- 修改4个标签名称和两组数据值

### 3. 箱线图
- 编辑3个工艺的最小值和最大值范围
- 动态生成模拟数据

### 4. 折线图
- 编辑X轴标题、Y轴标题、数据集标题
- 修改7个标签名称和两组数据值

### 5. 直方图
- 编辑X轴标题、Y轴标题、数据集标题
- 修改10个标签名称和数据值

### 6. 饼图/环形图
- 饼图和环形图切换功能
- 修改4个标签名称和数据值

### 7. 气泡图
- 编辑3个城市的X、Y坐标和气泡大小

### 8. 堆积面积图
- 编辑6个月的自然流量和广告投放数据

### 9. 误差棒图
- 编辑5个时间点的平均强度和误差范围

### 10. 雷达图
- 编辑2个产品在5个维度的评分数据

## 技术栈

- **前端框架**：纯HTML/CSS/JavaScript
- **图表库**：Chart.js、Plotly.js
- **样式**：CSS Grid、Flexbox、渐变背景
- **响应式**：媒体查询适配移动设备
